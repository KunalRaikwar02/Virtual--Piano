# 🎹 Virtual Piano (Interactive Web App)

An interactive virtual piano built using HTML, CSS, and JavaScript.  
Users can play piano sounds using keyboard keys or mouse clicks.

---

## 📌 Overview

This project simulates a real piano experience in the browser.  
Each key is mapped to a specific sound, allowing users to play music using their keyboard.

It focuses on:
- DOM manipulation
- Event handling
- Audio integration
- Interactive UI design

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---

## ✨ Features

- 🎹 Play piano using keyboard keys (A–L, W–P)
- 🖱️ Clickable piano keys
- 🔊 Real-time audio playback
- ⚡ Smooth key press animations
- 🎨 Clean and modern UI
- ✨ Hover animation for key labels
- 🎯 Responsive centered layout
